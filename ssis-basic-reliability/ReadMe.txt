These demonstrations assume that you have installed the AdventureWorks2008R2 and AdventureWorksDW2008R2 databases
in the local SQL Server 2008 R2 database instance. You can download the databases
available from http://msftdbprodsamples.codeplex.com.

In addition, you must extract the contents of the ssis-basic-data.zip file to the c:\ drive. This zip file is found in the found in the ssis-basic-reliability\begin folder.
